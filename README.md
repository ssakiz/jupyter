# jupyter by using docker


## 1- Installation

\# Buid docker image
```bash
docker build . -t jupyter/notebook
```

\# Run docker container 
```bash
docker run --rm -it -p 8888:8888 -v /home/burcu/jupyter/workdir:/home/jupyter jupyter/notebook
```
